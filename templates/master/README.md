# Reference Templates (Internal Only — NOT for Store Sale)

These PDF files are **reference templates and drawing patterns** sourced from
กรมโยธาธิการและผังเมือง (Department of Public Works and Town Planning) sample plans.

## Purpose

- Train and guide AI on professional drawing standards (line weights, title blocks, sheet layout, symbols)
- Provide technical structure patterns for permit-ready A3 output
- Inform the rule engine and validator — **not** to be copied or resold as-is

## What we do NOT do

- **Do not sell** these government sample plans on the Planasia Store
- **Do not list** template PDFs as marketplace products
- **Do not redistribute** raw template files to customers

## What the Store sells

The Store sells **original house designs co-created by users and AI**, using these
references only as technical guidance for layout and compliance patterns.

Each Store listing is tagged `source: community-ai` and tied to a user session.
